# 游戏规则

*   玩家指定一个数，代表长度，比如3 或 4.
*   电脑随机生成一个指定长度的整数
*   玩家输入一个数字
*   电脑判断玩家输入的数字是否正确
*   如果正确，游戏结束
*   如果不正确，电脑提示玩家输入的数字有几个数字和位置对（用数字+绿灯表示），几个数字对，但位置不对（用数字+黄灯表示）
    *    当鼠标移动到数字或绿灯、黄灯上时，显示tip，解释绿灯和黄灯含义。  
*   玩家继续输入数字，重复步骤3-6，直到猜对数字

## 示例1

1. 玩家指定长度为4
2. 电脑生成随机数字 1234
3. 玩家输入 1234
4. 电脑判断玩家输入的数字正确，游戏结束。

## 示例2

1. 玩家指定长度为4
2. 电脑生成随机数字 1234
3. 玩家输入 1235， 其中123数字和位置都对，5数字不对，电脑告诉玩家，3个位置对，显示3绿灯
4. 玩家输入 9235， 其中23数字和位置对，95数字不对，电脑告诉玩家，2个位置对，显示2绿灯
5. 玩家输入 5231， 其中23数字和位置对，1数字对，位置不对，5数字不对，电脑告诉玩家，2个位置对， 1个数字对，显示2绿灯1黄灯
6. 玩家输入 1234， 电脑判断玩家输入的数字正确，游戏结束。

# 登录
*   玩家可以以Guest身份玩游戏，也可以注册用户和登录玩。
*   游戏支持Google Auth登录。  
*   玩家登录后，游戏会显示Leader board。Leader board 显示所有玩家的成功通关次数，从大到小排列。
*   Leader board仅记录登录玩家，不记录Guest玩家。
*   玩家第一次登录游戏，数据库内没有该玩家记录时，弹窗提示玩家设置Profile。Profile包含一下选项：
    *   游戏名：游戏第一次会提供一个随机游戏名，并提示玩家修改。如果玩家没有修改，则使用该随机游戏名。
    *   选择是否记录游戏统计，并加入Leaderboard。缺省为选中。
*   点击Leader board内任一玩家，会显示更详细统计，比如通关3个数字，4个数字，5个数字次数，以及平均猜测步数（3个，4个，5个数字）。
*   增加登录玩家Profile页面，登录玩家可修改游戏名，是否参与Leaderboard，等。
    *   如果玩家选择不加入Leaderboard，删除该玩家之前的游戏记录，并从Leader board中删除。保留该玩家Profile。


# 游戏房间
*   玩家可以选择自己玩，也可以选择创建游戏房间，或加入现有游戏房间。
*   游戏房间功能仅限于登录玩家。
*   Guest玩家在主页面会看到游戏房间选项，但该选项Grey out，同时提示玩家需要登录可以开启游戏房间功能。
*   玩家登录后，游戏底部会显示已存在的房间。
*   玩家可以选择加入一个房间，或创建房间，或直接启动游戏自己玩。创建房间时可以命名。
*   每个房间最少1人，最多3人。房间内玩家可以看到处于房间的其他玩家。
*   房间创建者可以随时选择启动游戏。
*   房间内游戏启动后，随机选择一个玩家开始猜数字，之后round robin轮流猜测数字。每次猜测结果对房间内所有玩家均可见。
*   每次猜测有1分钟时间。如没有猜测，则轮到下一玩家猜测。
*   第一个猜中数字的玩家获胜。
*   每次游戏结束后，玩家可以选择退出，或继续。房间创建者可以选择启动游戏。
*   房间创建者退出房间是，房间关闭并删除。
*   在房间获胜的次数计入该玩家成功通关次数，加入Leader board。同时Leader board统计玩家在所有房间中的获胜次数，单独显示。
*   玩家如加入房间，游戏网页需定期通知服务器，Keep alive。玩家如关闭网页，自动通知服务器断开。或服务器超过2分钟没有收到玩家Keep alive信号，则断开连接，玩家自动退出房间。


# AI 任务
编写上述网页游戏，要求如下：
*   界面简洁，易于使用
*   代码清晰，易于维护
*   游戏逻辑正确，符合上述规则
*   游戏可以在浏览器和手机上运行
*   游戏前端用Angular， typescript编写。后端用Node.js + NestJS + typescript
*   在Angular基础上，包装成Android和IOS应用
*   使用Redis数据库 + AOF
*   使用Nginx作为reverse proxy

# 发布
*   使用Docker compose发布，创建三个容器：web app包括前端和后端，redis, Nginx。
    *   redis用AOF保存数据。
*   整个系统运行在Public VPS, 域名guessnumber.flamebots.org
*   Android应用发布在Google Play上
*   IOS应用暂时不发布，可以本地运行
