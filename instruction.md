# 游戏规则

1. 玩家指定一个数，代表长度，比如3 或 4.
2. 电脑随机生成一个指定长度的整数
3. 玩家输入一个数字
4. 电脑判断玩家输入的数字是否正确
5. 如果正确，游戏结束
6. 如果不正确，电脑提示玩家输入的数字有几个数字和位置对（用数字+绿灯表示），几个数字对，但位置不对（用数字+黄灯表示）
7. 玩家继续输入数字，重复步骤3-6，直到猜对数字

## 示例1

1. 玩家指定长度为4
2. 电脑生成随机数字 1234
3. 玩家输入 1234
4. 电脑判断玩家输入的数字正确，游戏结束。

## 示例2

1. 玩家指定长度为4
2. 电脑生成随机数字 1234
3. 玩家输入 1235， 其中123数字和位置都对，5数字不对，电脑告诉玩家，3个位置对，显示3绿灯
4. 玩家输入 9235， 其中23数字和位置对，95数字不对，电脑告诉玩家，2个位置对，显示2绿灯
5. 玩家输入 5231， 其中23数字和位置对，1数字对，位置不对，5数字不对，电脑告诉玩家，2个位置对， 1个数字对，显示2绿灯1黄灯
6. 玩家输入 1234， 电脑判断玩家输入的数字正确，游戏结束。

# 辅助功能
*   玩家可以以Guest身份玩游戏，也可以注册用户和登录玩。
    *   游戏支持Google Auth登录。  
    *   玩家登录后，游戏会显示Leader board。Leader board 显示所有玩家的成功通关次数，从大到小排列。
    *   点击Leader board内任一玩家，会显示更详细统计，比如通关3个数字，4个数字，5个数字次数，以及平均猜测步数（3个，4个，5个数字）。
